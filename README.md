A sleek, modern portfolio website built to showcase my skills, projects, and experience as an IT student and aspiring developer. Designed with a focus on **premium aesthetics**, **smooth animations**, and **responsive layouts** across all devices.

| Feature | Description |

This site is deployed on **Vercel** with automatic deployments. To update:

```bash

git commit -m "Your update message"
git push
```

Vercel will automatically re-deploy your site within seconds! ⚡
